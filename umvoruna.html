Stormlight World Guide

The Stormlight World Guide is a world guide for Roshar.
The book contains information about the magic systems, highstorms, plants, wildlife, cultures, as well as a variety of other things that can be found on Roshar.

Stormlight Handbook

The Stormlight Handbook is a game guide for the players for the systems on Roshar.
The book contains information about the game system in general as well as for specific systems that can only be found on Roshar.

Stonewalkers Adventure

The Stonewalkers Adventure is a story module set on Roshar.
The book contains a fully finished adventure that gamemasters can use out of the box or change to their liking, useful for new and experienced gamemasters alike.

Mistborn World Guide

The Mistborn World Guide is a world guide for Scadrial.
The book contains information about the magic systems, mist, plants, wildlife, cultures, as well as a variety of other things that can be found in both eras on Scadrial.

Mistborn Handbook

The Mistborn Handbook is a game guide for the players for the systems on Scadrial.
The book contains information about the game system in general as well as for specific systems that can only be found on Scadrial.

Legacy Adventure

The Legacy Adventure is a story module set on Scadrial.
The book contains a fully finished adventure that happens during both eras on Scadrial that gamemasters can use out of the box or change to their liking, useful for new and experienced gamemasters alike.
